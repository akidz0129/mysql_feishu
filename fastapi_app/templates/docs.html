GET /a

获取某个客户的订单记录。

参数:

  • customer_id (string): 客户唯一 ID（必填）
{
  "customer_id": "12345",
  "orders": [
    {"order_id": 1001, "amount": 320.5, "date": "2024-06-01"}
  ]
}

GET /b

参数:

  • category (string): 商品分类，如 "fruit"
{
  "category": "fruit",
  "trend": [
    {"date": "2024-06-01", "sales": 200}
  ]
}
